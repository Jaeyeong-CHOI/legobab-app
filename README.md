## 💡 Why LegoBab?

DGIST 캠퍼스는 성장하고 있습니다. 학생 1,200명 이상, 반도체공학과 신설, 대학원 확장. 하지만 E7 식당가는 여전히 **줄서기 15분**, 외국인 학생 15%의 **할랄·비건 수요**에 대응하지 못하고 있습니다.

> *"캠퍼스가 글로벌해지면, 한 끼도 글로벌해져야 한다"*

레고밥은 **레고처럼 반찬 블록을 조립하는 맞춤 도시락 앱**입니다.  
다국적 식이 필터, 스마트 로커 무인 픽업, 친환경 용기 순환 시스템을 통해 캠퍼스 식문화를 혁신합니다.

## 🎯 6대 핵심 축

| 축 | 설명 |
|---|---|
| 🧱 **레고 조합** | 베이스 7종 + 반찬 블록 20종+ 자유 조합 |
| 🌍 **다국적 필터** | 할랄 · 비건 · 글루텐프리 · 알레르기 + 한/영/힌디 다국어 |
| 🏥 **디지털 헬스케어** | 실시간 칼로리·탄단지 분석 + 균형 점수 + 주간 트래커 |
| ♻️ **친환경 순환** | 레고형 수거함 반납 + 다회용기 보증금 + 그린포인트 + 탄소 시각화 |
| 👥 **같이밥** | 그룹 오더 최대 15% 할인 + 캠퍼스 네트워킹 |
| 🎮 **레고 러시** | 저녁 잔여 블록 룰렛 타임세일 → 음식 폐기 제로 |

| 👤 **마이페이지** | 프로필, 주간 트래커, 그린포인트, 식이 설정, 다국어 전환 |

## 🤖 운영 모델

```
① 조리   →  사장님이 블록별 반찬 대량 조리
② 적재   →  스마트 로커에 세팅 (냉장/온장)
③ 주문   →  학생이 앱에서 블록 조합 → 결제
④ 픽업   →  QR 스캔 → 로커 자동 오픈 → 수령
⑤ 반납   →  캠퍼스 레고 수거함에 용기 조립 반납
```

### 비용 비교

| 항목 | 기존 도시락집 | 레고밥 |
|---|---|---|
| 인건비 | 사장 + 알바 2~3명 | 사장 1명 |
| 대기 시간 | 10~15분 | 0분 (QR 즉시) |
| 가격 | 5,000~6,000원 | 3,500~4,500원 |
